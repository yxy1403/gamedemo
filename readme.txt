绿野迷踪 (Green Wild Mystery)
📖 项目简介
《绿野迷踪》是一款基于 HTML5 Canvas​ 开发的 Roguelike 生存游戏。玩家需要在充满怪物和敌对人类的废弃区域中搜集物资、击败敌人、强化自身，并最终成功撤离。

本项目旨在验证 “人类敌人 AI 行为逻辑”​ 与 “收集-养成闭环系统”​ 在轻度生存游戏中的可行性。
✨ 核心玩法
探索与搜刮：在随机生成的环境中搜索宝箱、木桶和容器，寻找武器、护甲和补给品。
动态战斗：面对普通怪物、精英怪以及拥有多阶段技能的 Boss。
人类威胁：独特的 “寻金者/探险家”​ NPC 系统，他们会主动拾取地上的物资，并在受到威胁时反击或逃跑。
撤离结算：击败 Boss 获得撤离钥匙，在倒计时结束前逃离，将战利品存入仓库。
🛠️ 技术实现与亮点
1. AI 工具实践 (Trae)
本项目全程使用 Trae (AI Coding)​ 辅助开发：
利用自然语言指令快速生成游戏基础骨架（Canvas 渲染、碰撞检测框架）。
通过 AI 辅助编写复杂的 UI 交互逻辑和配置文件。
人工介入进行核心逻辑优化、性能调优及算法实现。
2. 性能优化：四叉树 (QuadTree) 算法
为了解决高同屏实体（子弹、怪物、物品）下的性能瓶颈，摒弃了传统的 O(n²) 暴力遍历，引入了 四叉树空间分割算法。
效果：将碰撞检测的复杂度降至 O(n log n)，确保在 100+ 实体同屏时依然保持 60 FPS 流畅运行。
3. 复杂 AI 行为设计
怪物 AI：包含领地巡逻、追击、攻击、击退及无敌帧机制。
人类 AI (状态机)：实现了复杂的决策逻辑，NPC 会根据当前状态（游荡 / 追击玩家 / 拾取物资 / 逃跑）动态切换行为。
4. 数据驱动配置系统
所有游戏数值（武器属性、护甲加成、掉落概率、Boss 技能）均通过 config.js进行集中管理，实现了：
武器系统的差异化武器。
RPG 成长：背包扩容、仓库存储、图鉴收集（收集品转化为永久属性加成）。
